Project Title: Mini File Search Utility (Bash Script in Unix)

Meaning / Description:

The Mini File Search Utility is a menu-driven Bash script designed for Unix/Linux systems that allows users to quickly and efficiently search files within a directory. It uses standard Unix commands like find, grep, and optionally Git commands to locate files based on different parameters such as name, extension, content, size, or last modification date.

This utility simplifies file management tasks, helping users avoid long and complex command-line searches. It provides an interactive, user-friendly interface suitable for students, developers, and system administrators.

Main Features:

-Search by File Name

Finds files matching an exact name in the given directory.

-Search by File Extension

Lists all files with a specific extension (e.g., .txt, .sh, .cpp).

-Search by Content

Uses grep to locate files containing specific text or keywords.

Search by File Size
Finds files larger or smaller than a given size (e.g., +1M, -500k).

Search Files Modified in Last N Days
Displays files recently changed or updated.

Menu-Driven Interface
Easy-to-use and continuously loops until the user chooses to exit.

Optional Git Integration (Advanced Feature)
Allows searching tracked files, viewing commit logs, and checking the last modification made to a file.

Exit Option
Gracefully ends the program when the user chooses to exit.
